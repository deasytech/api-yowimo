# Claude Code Instructions

When contributing to Yowimo:

1. Respect the long-term architecture.
2. Avoid short-term hacks that create long-term debt.
3. Build reusable systems.
4. Prioritize maintainability over speed.
5. Think globally.
6. Design for millions of users.
7. Keep documentation current.
8. Every major feature should move Yowimo closer to this vision.
