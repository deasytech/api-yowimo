# Claude Code Instructions

When implementing roadmap initiatives:

1. Prioritize foundational architecture before advanced features.
2. Build reusable systems.
3. Avoid premature optimization.
4. Validate assumptions through metrics.
5. Keep documentation synchronized with roadmap progress.
6. Design for global scale.
7. Consider operational impact before implementation.
8. Update this roadmap at the end of every quarter.

# Engineering North Star

Every technical decision should answer

- Does it improve the player experience?
- Does it scale globally?
- Is it maintainable?
- Is it secure?
- Is it measurable?
- Can it evolve without major rewrites?
- Does it strengthen the creator economy?
- Does it move Yowimo closer to its long-term vision?

If the answer to most of these is "yes," the initiative aligns with the platform roadmap.
